Despre Aplicatie

DeviceRec este o aplicatie web care permite utilizatorilor sa caute, filtreze si vizualizeze produse electronice. Sistemul include functionalitati precum:

  • Cautare in timp real dupa cuvinte cheie
  • Filtrare dupa tip, culoare si pret
  • Autentificare si vizualizare personalizata
  • Panou de administrare pentru adaugare si editare produse
  • Scraping in fundal de pe site-uri ca Newegg si eMAG

Tehnologii utilizate

  • Frontend: HTML5, CSS, JavaScript Vanilla
  • Backend: Node.js (http module nativ)
  • Baza de date: SQLite
  • Web scraping: Puppeteer

Structura aplicatiei

Sistemul este compus din mai multe componente:

  • Interfata vizuala (frontend) – permite interactiunea cu utilizatorul
  • Server backend – gestioneaza sesiuni, cautari, recomandari
  • Cache in memorie – retine produsele populare
  • Scraper – colecteaza automat date din surse externe
  • Baza de date SQLite – persistenta informatiilor

Codul sursa este disponibil pe GitHub.
